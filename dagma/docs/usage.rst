Install the package
-------------------

We recommend using a virtual environment via ``virtualenv`` or ``conda``, and use ``pip`` to install ``dagma``.
To use DAGMA, first install it using pip:

.. code-block:: console

   (venv) $ pip install --upgrade dagma
